## Enhanced World Creation

This mod expands the world creation screen and adds settings for changing its default 
parameters.

![preview](https://cdn.modrinth.com/data/cached_images/f205ad5f695cb4c84de991b06fb0467c543f87b4.jpeg)

## Usage

A new button will appear during world creation. Clicking it will open a screen where you can add an icon for the world and install a built-in resource pack. This resource pack will be automatically applied to the player upon entering the world.

![preview](https://cdn.modrinth.com/data/cached_images/e44d408195a84f9d65b2169c760a72baceb35d2a.jpeg)

The mod has a configuration option. In its settings, you can set the default parameters for world creation. This will significantly speed up the process if you frequently generate new worlds.

![preview](https://cdn.modrinth.com/data/cached_images/ec79550b6606e0602e33e8e66a1dbfd9dffdba89.jpeg)

I don't plan to add default game rules or data packs. Mods already exist for that.
